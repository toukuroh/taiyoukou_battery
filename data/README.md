# EA-20 気象データの置き場所

## データの取得方法

1. NEDO METPV-20 サイトにアクセス
   https://www.nedo.go.jp/library/nissharyou.html

2. 地点を選択して「標準年」または「寡照年」のデータをダウンロード

3. ダウンロードしたCSVを `ea20.csv` にリネームしてこのフォルダに配置

## 対応フォーマット

src/nedo_reader.py は以下の列名を自動認識します:

| 列の内容 | 認識するキーワード |
|---------|-----------------|
| 全天日射量 (GHI) | ghi, 全天, gh |
| 直達日射量 (DNI) | dni, 直達, dn |
| 散乱日射量 (DHI) | dhi, 散乱, diffuse, dh |
| 外気温 | temp, 気温, tair, ta |
| 相対湿度 | rh, 湿度, humidity |
| 風速 | ws, 風速, wind |
| 月 | month, 月 |
| 日 | day, 日 |
| 時 | hour, 時 |

## 単位について

- 日射量が [MJ/m²/h] の場合は自動的に [W/m²] に変換されます
  (1 MJ/m²/h ≈ 277.8 W/m²)
- 日射量が [W/m²] の場合はそのまま使用されます

## サンプルデータ

実際のデータがない場合、`main.py` 実行時に東京の近似データが
自動生成されます（テスト用）。
